ATM系统应用入门教程

了解Lifecycle的一个很好的方式是学习这个入门教程，它将告诉你如何使用Lifecycle框架来增强你的应用。这个应用将构建一个ATM系统，给客户提供存钱和取钱服务。

跟随这个入门教程来了解Lifecycle如何使得开发更简单和更健壮。

  • 看样例如何分开状态管理逻辑和业务逻辑

当你完成入门教程你将能够：

  • 创建一个使用Lifecycle框架的应用。

开始

下文将介绍如何配置你的本地机器来进行开发。 如果你只是想看入门教程，你可以直接去看第一步： 1 - 问题定义。

安装 Git

你可以下载和安装Git从http://git-scm.com/download。 一旦安装成功，你应该可以访问git命令行工具。 你将使用到的主要命令是：

  • git clone ... : 克隆一个远程仓库到你的本地机器。
  • git checkout ... : 切换到某个特别的分支或者标记的版本。

下载metafsm-atm

克隆位于GitHub的metafsm-atm 仓库，使用以下命令：

git clone  https://github.com/TracyLu/metafsm-atm.git

这个命令在你的当前目录下创建metafsm-atm 目录.

切换你的当前目录到metafsm-atm.

cd metafsm-atm

从现在起，这个入门教程的所有说明都是假定你运行所有的命令都是在 metafsm-atm目录下。

安装 JDK

首先，当你打算要基于Java来开发应用时，你必须安装JDK 1.7 +。

下载URL：  http://www.oracle.com/technetwork/java/javase/archive-139210.html.

安装完毕后，你可以运行以下命令来检查JDK的版本：

java -version

安装sbt

SBT是一个针对Scala和Java项目的、开源的构建工具，类似于Java的Maven和Ant。

在sbt的官网上有安装指南，请根据你的操作系统，安装最新的sbt版本。

目前，我们使用的是0.13.8版本。

下载URL：  http://www.scala-sbt.org/download.html

请运行以下命令检查你安装的sbt版本：

sbt -version

安装IntelliJ IDEA Community Edition

非常不错的IDE，支持Java,Scala等很多开发语言。因为我们在程序中使用了Scala语言，所以选择了IDEA。

你可以下载最新版本。目前，我们使用的是14.1.4版本。

下载URL：  http://www.jetbrains.com/idea/download/
